The Target Score is generated at random.

Clicking on a gem will add a specific number of points to your total score.

You win the game by matching Player Score to Target Score, you lose the game if Player Score exheeds Target Score.

The value of each gem is hidden until you click on it.

Each time a new game starts a new value will be given to each gem.
